# mdbook-frontmatter-strip

Strip YAML frontmatter from rendered HTML

## Installation

```bash
cargo install mdbook-frontmatter-strip
```

Tested against mdbook v0.5.1

Tested against rust editions 2020 & 2024

## Usage

```toml
[preprocessor.frontmatter-strip]
renderers = ["html"]
```
